Build a full-stack sportsbook web application called KialaBet inspired by modern African betting platforms like Betika and SportPesa, but with unique branding, UI, and codebase.

PROJECT GOAL:
Create an MVP sportsbook platform with demo betting functionality and disabled real payments for now.

TECH STACK:
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL
- ORM: Prisma
- Authentication: JWT + bcrypt
- Realtime updates: Socket.io
- State management: Context API
- Styling: Tailwind CSS
- Icons: Lucide React

DESIGN:
- Premium dark mode interface
- Main colors:
   - dark background (#0B0F19)
   - green accent (#00C853)
   - white text
- Mobile-first responsive design
- Smooth animations
- Clean sportsbook cards
- Sticky mobile bottom navigation
